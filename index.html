Introduction

It is the interface between javascript and html+css.
This is how we can control the interactivity on a page using the javascript we have learnt.

First we need to understand how the world of web works:

  • Every html file that we created is converted into one big javascript object called document when it needs to be executed. Every tag is another object in the object document.
  • There are a vast number of properties in dom that we can set like mousewheel and hover and many more. It is not possible to learn all of them. We will learn the important one's and will learn about the others as we encounter them.
  • Dom works in a very simple way.

Select and Manipulate

Selecting an object:
var h1=document.querySelector("h1");

Manipulating an object:
h1.style.color="pink";

Different ways of selecting an object in javascript

Some of the important one's are:
document.getElementById("...");
 document.getElementsByClassName("...); //The variable using this is a list of the elements having the class name
 document.getElementsByTagName("..."); //List of the elements
 All this can be done by using query selector meaning all the selectors in css can be used here but query selector returns only the first element that matches the value.

Manipulating Attributes

We can change the attributes of a tag.
selector.getAttribute("attribute") //retrieves the value of the attribute
 selector.setAttribute("attribute","value") //Sets the value of the attribute

To change the value of text being displayed on the page we can use textcontent or innerHTML.
We already learnt what basic dom manipulation is and how to use them. It included changing styles, attributed or classes. But it was all happening when the page was loaded.
Now we will learn dom manipulation to make page interactive. We will learn how to do something when particulat event happens like a click or hover.

Eventlistener

This is achieved using an eventlistener that listens for an event on our selectors.
selector.addEventListener(type,functiontocall);

Type means the type of event like a "click". Functiontocall can be an anonymous function or some external function.
We can have multiple listeners on same element and they will be executed one after the other.
Inside of a listener the keyword this ,if used refers to the item that was clicked or event took place on.
